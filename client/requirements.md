## Packages
framer-motion | Smooth page transitions and layout animations
recharts | Financial visualization charts for league dues and payouts
date-fns | Formatting dates for transaction history and league timelines
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind classes safely

## Notes
- Authentication is handled via Replit Auth (use-auth.ts hook already exists)
- Payment integration is mocked on frontend but calls backend API
- "Sync Platform" simulates connecting to ESPN/Yahoo APIs
- Images use Unsplash for sports/stadium backgrounds
- Tailwind config needs font-display (Outfit) and font-mono (JetBrains Mono)
